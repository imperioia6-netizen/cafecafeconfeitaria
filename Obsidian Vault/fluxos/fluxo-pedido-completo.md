FLUXO COMPLETO DE PEDIDO (GERAL)

Ordem obrigatoria para QUALQUER tipo de pedido:

1. Boas-vindas + link do cardapio + perguntar tipo (delivery/encomenda/retirada)
2. Se tem pedido em aberto: perguntar se quer continuar ou novo
3. Entender o pedido do cliente
4. Confirmar cada item de forma curta ("Anotado!")
5. Perguntar: "Gostaria de mais alguma coisa?" — OBRIGATORIO
6. So avancar quando disser que nao quer mais nada
7. Coletar dados: nome completo, endereco (se entrega), data, horario
8. Confirmar tipo: retirada ou delivery (se ainda nao informou)
9. Calcular total completo
10. Informar sinal se necessario
11. Informar forma de pagamento
12. Resumir pedido UMA VEZ (no final)
13. Aguardar confirmacao/comprovante
14. Registrar na plataforma

REGRAS:
- NAO pular etapas
- NAO assumir informacao — perguntar
- NAO repetir lista de itens a cada mensagem — resumo so no final
- NAO perguntar "retirada ou delivery?" no meio — so no final
- SEMPRE validar antes de confirmar
