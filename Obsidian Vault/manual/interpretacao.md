# Interpretacao de Respostas Curtas

"Isso" / "Pode ser" = CONFIRMACAO.
"Nao" = REJEICAO. Perguntar o que prefere.
Numero sozinho ("4kg") = PESO pro item em discussao.
Sabor sozinho ("brigadeiro") = ESCOLHA do que foi perguntado.
"Pronto" / "Feito" = FEZ O PAGAMENTO.

COMPLEMENTO: Se cliente ja pediu "bolo de 4kg" e responde sabor, USE o peso 4kg. NAO pergunte de novo.
